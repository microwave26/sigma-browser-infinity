*A browser with nothing you need - Limited possibilities*

A low-quality web browser built with Python and PyQt5, featuring a sleek modern design, advanced privacy features, and powerful tools for the modern web.

![Sigma Browser Screenshot](screenshots/sigma-browser.png)

## ✨ Features

### Core Features
- 🌙 **Dark/Light Mode Toggle** - Switch between beautiful dark and light themes with one click
- 🕶️ **Private/Incognito Mode** - Browse without saving history, cookies, or site data
- 🛡️ **Built-in Ad Blocker** - Block ads and trackers automatically for faster browsing
- 📥 **Download Manager** - Pause, resume, and manage all your downloads in one place
- 🔐 **Password Manager** - Securely store and manage passwords with encryption
- 📁 **Tab Groups** - Save and organize tabs into groups for later use
- 🧩 **Extensions Support** - Enable/disable extensions to customize your experience

### Advanced Features
- 🔍 **Developer Tools** - Inspect elements, view source, and debug web pages
- ⌨️ **Keyboard Shortcuts** - Professional shortcut system for power users
- 🖨️ **Print Support** - Print web pages directly from the browser
- 🔒 **Encrypted Storage** - All passwords and sensitive data are encrypted
- 📂 **Tab Management** - Drag, drop, and organize tabs with ease
- 🎯 **Smart URL Bar** - Search Google or type URLs directly

## 🚀 Quick Start

### Prerequisites
- Python 3.7 or higher
- pip package manager

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/p4ncake3/sigma-browser-infinity.git
cd sigma-browser-infinity
```
2. **Install dependencies:**
```bash
pip install -r requirements.txt
```
3. **Run the browser**
```bash
python browser.py
```
**Optional: Enhanced security**
For stronger password encryption:
```bash
pip install cryptography
```
